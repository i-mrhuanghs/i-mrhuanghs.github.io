备份服务器数据库样例

Posted by     "hansen" on Friday, June 15, 2018

备份服务器数据库样例

mysqldump -ubumblebeeuser -hstg2mysql-1.czrdgd08qzk4.eu-north-1.rds.amazonaws.com -p -d bumblebee_bc > 20210615.bumblebee.sql
# 仅备份数据表结构
mysqldump -ubumblebeeuser -hstg2mysql-1.czrdgd08qzk4.eu-north-1.rds.amazonaws.com -p  bumblebee_bc > 20210615.bumblebee.sql
# 备份数据表数据和结构
ls -lh
# 查看备份下来的数据大小
tar -cjf b.tar.bz2 20210615.bumblebee.sql
# 网络慢的，压缩一下方便下载
logout
# 退出远程连接
scp username@stg2bumblebee-node-1.micai.local:b.tar.bz2 .
# 拷贝到本地，.指当前目录文件
